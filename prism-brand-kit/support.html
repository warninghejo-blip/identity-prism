Support

Use this page as the public help destination for any Prism app before app-store submission.

Support channels

  • Email: [support@your-domain]
  • Security reports: [security@your-domain]
  • Community / social: [Discord / Telegram / X]
  • Status page: [status.your-domain]

Suggested response targets

  • Critical account/security issue: within 24 hours
  • Billing / entitlement issue: within 1 business day
  • General product issue: within 2-3 business days

What users should send

  • Public wallet address only — never private keys or seed phrases.
  • Timestamp, network, and action that caused the issue.
  • Order ID / receipt / entitlement reference if billing is involved.
  • Screenshots or transaction signature where relevant.

Security escalation wording

Never ask users to send private keys, seed phrases, or wallet recovery codes. Any real support flow under the Prism brand should repeat this warning.

Recommended support categories

  • Wallet connection
    Connection issues, mobile wallet adapter, wallet permissions, disconnect/reconnect.
  • Transactions and signing
    Failed signing flow, suspicious transaction warnings, relay/simulation behavior.
  • Billing, PRISM, and subscriptions
    PRISM purchases, passes, vault allocation, subscription redemption, payment issues.
  • Account or content issues
    Profile data, generated outputs, sync delays, eligibility questions.
