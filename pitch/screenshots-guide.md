# Screenshot Guide for Pitch Video

Все скриншоты делать при подключённом кошельке с хорошим скором.
Браузер: fullscreen (F11), разрешение 1920x1080.

---

## Screenshot 1: Celestial Card (ГЛАВНЫЙ скриншот)
**URL:** `https://identityprism.xyz/`
**Что сделать:**
1. Подключи кошелёк с хорошим скором (>500)
2. Дождись загрузки hub-меню
3. Нажми на карту (она откроется)
4. Скриншот ЛИЦЕВОЙ стороны — планета, score ring, tier name, badges

**Что должно быть видно:** Светящаяся карточка с планетой, круглый индикатор скора, tier (например "Quasar"), бейджи внизу.

**Для Veo:** Это start frame для Segment 3 (Product). Самый красивый и впечатляющий визуал.

## Screenshot 2: Composite Score Breakdown
**URL:** `https://identityprism.xyz/`
**Что сделать:**
1. Открой карту (клик)
2. Промотай вниз или переверни карту (клик по ней)
3. Найди секцию "Composite Score" с 5 полосками
4. Раскрой 2-3 полоски (клик по ним) чтобы были видны детали
5. Скриншот

**Что должно быть видно:** 5 цветных progress bars (On-Chain, Sybil Trust, Human Proof, Social, Engagement) с раскрытыми деталями — формулы подсчёта, Badge Bonus.

**Для Veo:** End frame для Segment 3 — показывает глубину аналитики.

## Screenshot 3: CosmicHub (главное меню)
**URL:** `https://identityprism.xyz/`
**Что сделать:**
1. Подключи кошелёк
2. Не открывай карту — останься на hub-экране
3. Скриншот: видны тайлы меню (Forge, Arena, Scan, Games, Quests, Vault, Leaderboard)

**Что должно быть видно:** Космический фон со звёздами, мини-паспорт вверху (аватар + скор + tier), сетка навигационных тайлов с иконками.

**Для Veo:** Можно использовать как промежуточный кадр, показывающий breadth продукта.

## Screenshot 4: Game Mode (PrismLeague)
**URL:** `https://identityprism.xyz/game`
**Что сделать:**
1. Открой страницу Games
2. Выбери Orbit Runner или Gravity Runner
3. Начни игру и сделай скриншот В ПРОЦЕССЕ игры (корабль летит, астероиды вокруг)
4. Или: скриншот экрана выбора игр (5 карточек с баннерами)

**Что должно быть видно:** Геймплей с космическим кораблём, или карусель из 5 игровых режимов с красивыми баннерами.

**Для Veo:** Показывает "gamified" часть — Human Proof через игры. Лучший start frame для мини-демо игр.

## Screenshot 5: Leaderboard + Tournaments
**URL:** `https://identityprism.xyz/leaderboard`
**Что сделать:**
1. Открой Leaderboard

**Что должно быть видно:** Таблица лидеров, вкладки (Overall/Orbit/Destroyer/Gravity/Tournament), подвкладки Daily/Weekly/Monthly, информация о призовом фонде.

**Для Veo:** Показывает competitive + economy layer.

## Tips
- Делай скриншоты на мониторе 1920x1080 (или 2560x1440 и ужми)
- Тёмная тема уже стоит по дефолту — ничего менять не надо
- Для красоты: подключи кошелёк с максимальным скором и бейджами
- Если нет кошелька с хорошим скором — используй тестовый с сервера
- Скриншоты можно делать через DevTools (Ctrl+Shift+P → "Capture full size screenshot")
